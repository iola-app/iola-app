# Messenger App

## Install
- `yarn`
- `react-native link`

## Development
1) Run react-native server + GraphQL scheme watch
  - `yarn start`
2) Run your emulator:
  - Android:
    - `yarn run:android`
    - `yarn log:android`
  - iOS:
    - `yarn run:ios`
    - `yarn log:ios`
3) `yarn postinstall` // @TODO: remove when Apollo fix this: https://github.com/apollographql/apollo-client/issues/3236
